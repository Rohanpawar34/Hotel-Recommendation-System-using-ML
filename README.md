# Hotel-Recommendation-System-using-ML
A machine learning-based hotel recommendation system that suggests hotels based on user preferences such as location, price, ratings, amenities, and reviews.

## Features
- Personalized hotel recommendations
- Similar hotel suggestions
- Budget filtering
- Rating filtering
- Amenities filtering
- Customer review sentiment analysis
- Hotel booking system
- Analytics dashboard
- Interactive visualizations
- Modern Streamlit UI

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
- TextBlob
