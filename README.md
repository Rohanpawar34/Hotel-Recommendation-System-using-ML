# Hotel-Recommendation-System-using-ML
A machine learning-based hotel recommendation system that suggests hotels based on user preferences such as location, price, ratings, amenities, and reviews.

## Features
* Personalized hotel recommendations
* Similar hotel suggestions
* Budget, rating, and amenities filtering
* Customer review sentiment analysis using TextBlob
* Hotel booking system
* Interactive analytics dashboard
* Modern Streamlit user interface

## Technologies Used
* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Plotly
* TextBlob

## Machine Learning Techniques

* Content-Based Recommendation (TF-IDF + Cosine Similarity)
* Sentiment Analysis using TextBlob
* K-Means Clustering for hotel categorization


## Dashboard insights
The application provides interactive insights, including:
* Hotels by city
* Average hotel prices
* Hotel ratings by category
* Top-rated hotels
* Hotel availability statistics
